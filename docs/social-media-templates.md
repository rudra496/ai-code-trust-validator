## LinkedIn Post

I just shipped v0.4.0 of AI Code Trust Validator — an open-source tool that catches security flaws, AI hallucinations, and logic errors in code generated by ChatGPT, Claude, Copilot, and other AI tools.

Why I built this:
84% of developers use AI coding tools, but only 29% trust the output. I was in the 29% — and for good reason. AI code often contains fake imports, security vulnerabilities, and subtle logic errors that look fine in code review but break in production.

What it does:
- Analyzes Python, JavaScript, and TypeScript files
- Trust score (0-100) with detailed breakdown
- AI-powered auto-fix using OpenAI, Anthropic, or Ollama
- VS Code + JetBrains plugins, GitHub Action, Docker, REST API

Install: pip install ai-trust-validator

Open source (MIT) and looking for contributors!
